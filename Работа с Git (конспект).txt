1) Для начала работы в Git пишем:

git init

2) Посмотреть состояние проекта:

git status


3) Для указания какие файлы сейчас добавляем в коммит:

git add .  (добавить всё)
git commit
Откроется VIM, чтобы добавить комментарий нажимаем insert, чтобы выйти жмём esc :wq enter

git add <file>  (добавить конкретный файл)

Второй способ коммитить:
git commit -m "Коммит 2. Добавил style.css"

4) Дла просмотра имеющихся коммитов:

git log

5) Для переключения между коммитами:

git checkout <хэш коммита>

6) Чтобы перейти к самому последнему коммиту:

git checkout master

7) Для создания новой ветки:

git branch <имя ветки> (! только создаёт, не переключает на новую ветку)

8)Для переключения между ветками:

git checkout <имя ветки>

9) Для слияния веток:

переходим в ветку master, вводим 
git merge <имя ветки>

10) Для восстановления состояния проекта до последнего состояния:

git reset --hard

git revert HEAD --no-edit

11) Посмотреть всю историю Git:

gitk


12) Добавить удалённый репозиторий (https://github.com/jimfoxes/git-examle):

git remote add

13) Посмотреть удалённые репозитории:g

git remote -v

14) Отправить коммиты в удалённые репозитории:

git push -u <имя удаленного репозитория> --all

15) Продолжить работу на другом компьютере:

git clone <ссылка на удалённый репозиторий>

16) Перед началом работы, если есть работа с проектами на разных компах:

git pull

17) git fetch получение изменений с сервера
